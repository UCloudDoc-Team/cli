# restart

\#\# ucloud uhost restart

Restart uhost instance

\#\#\# Synopsis

Restart uhost instance

\`\`\` ucloud uhost restart \[flags\] \`\`\`

\#\#\# Examples

\`\`\` ucloud uhost restart --uhost-id uhost-xxx1,uhost-xxx2 \`\`\`


``` 
--zone     string            Optional. Assign availability zone

```
